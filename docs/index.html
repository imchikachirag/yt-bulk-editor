YouTube Bulk Editor

Edit titles, descriptions and tags for all your YouTube videos in one place. No more opening each video one by one.

We request YouTube management permission to read and update your videos.
Your token never touches our server after login. Read the source code.

How to use this tool
  1. 1Sign in with Google using the account that manages your YouTube channel. Brand Account users: sign in as the personal account that owns the brand account.
  2. 2Select your channel if you manage multiple YouTube channels.
  3. 3Edit inline - click any Title, Description or Tags field to edit it. Changed fields turn yellow and show the original value above for reference.
  4. 4Save row by row with the blue Save button, or use Save All in the header to push every pending change at once.
  5. 5Sort and filter using the toolbar dropdowns. Use pagination to browse pages of 10, 25 or 50 videos.
  6. 6Export CSV at any time for a full backup of all your video metadata.
  7. 7Disconnect when done. This revokes your OAuth token at Google immediately.

How It Works - Architecture and Privacy

This tool is built entirely on trusted, industry-standard infrastructure. Every component is auditable. The full source code is on GitHub.

1
You sign in
Clicking "Sign in with Google" redirects you to Google's own login page. We never see your password.
Google OAuth 2.0
2
Login handshake
Google sends a one-time code to our server to complete the secure handshake. This is standard OAuth - required by Google's own protocol.
Google Cloud Run (by Google)
3
Token to your browser
Our server exchanges the code for a token and immediately sends it to your browser. The token is never stored, logged, or written to any database.
sessionStorage - browser only
4
Direct to YouTube
All video reads and edits go directly from your browser to YouTube. Our server is never involved again. Your video data never passes through us.
YouTube Data API v3 (by Google)
Your OAuth token is cleared from sessionStorage the moment you close this tab or click Disconnect - which also revokes it at Google. Nothing persists. The source code proving this is publicly readable at github.com/imchikachirag/youtube-bulk-editor
How to Use
  1. 1Sign in with Google using the account that manages your YouTube channel. Brand Account users: sign in as the personal account that owns the brand account, then select the brand channel from the picker.
  2. 2Edit inline - click any Title, Description or Tags field to edit it. Changed fields turn yellow and show the original value above for reference.
  3. 3Save row by row with the blue Save button, or use Save All in the header to push every pending change at once.
  4. 4Sort and filter using the toolbar. Sort by title or date. Filter to show only changed or saved rows. Use pagination for large channels.
  5. 5Export CSV at any time for a full backup of all video metadata.
  6. 6Disconnect when done. This revokes your OAuth token at Google immediately.
