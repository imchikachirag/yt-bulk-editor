What Changed in v2.3.3
FIXED
Import CSV: comparison trims both sides Description and title comparisons now trim whitespace on both the CSV value and the in-memory value. YouTube descriptions with trailing newlines previously caused all rows to falsely appear as changed even when the user had not edited them.
FIXED
Import CSV: clear message when no changes found When no actual changes are detected in the imported file, a clear amber message now explains this. Previously a modal appeared with all rows showing "No Change" and a disabled button, giving the impression that nothing happened.
FIXED
Import CSV: FileReader wrapped in try/catch with error handler The file reader now has a try/catch and an onerror handler so any parsing failures surface as readable messages rather than silently failing with no feedback.
FIXED
Contact link in footer: replaced Cloudflare email obfuscation with mailto The Contact link in both footer instances and the feedback form note used Cloudflare email obfuscation which only works behind a Cloudflare proxy. Replaced with direct mailto links.
Bugs Fixed in v2.3.1
FIXED
CORS: exact origin match (security hardening) CORS origin check now uses exact string equality instead of prefix matching.
FIXED
importParsed undeclared in strict mode The import state variable was used without a top-level declaration. In strict mode this is a ReferenceError.
FIXED
Refresh stays on selected channel (multi-channel fix) Refresh button now correctly reloads the channel you had selected instead of always defaulting to the first.
FIXED
rel=noopener noreferrer on dynamic video links Dynamically generated video thumbnail links now include rel="noopener noreferrer".
Feature Added in v2.3.0
NEW
Export Visible rows The Export button is now a split button. Export All downloads the full channel backup. Export Visible downloads only the rows currently shown after any search or filter. The filename includes -filtered so you can tell them apart.
Bugs Fixed in v2.2.1
FIXED
Dark mode: text invisible while typingEditable textareas had hardcoded white backgrounds for hover and changed states.
FIXED
Dark mode: row flashes green on hoverA generic tr:hover td rule conflicted with the specific saved-row hover rule.
FIXED
Import CSV: preview modal not appearingCSV headers now have surrounding quotes stripped, fixing Excel-saved file compatibility. Error toasts stay 5 seconds.
FIXED
Save: cryptic error with angle bracketsApp now validates for < and > before calling the YouTube API and shows a clear message.
