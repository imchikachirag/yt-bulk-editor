Privacy Policy

Last updated: May 2026  ·  Effective immediately  ·  by Chirag Mehta

The short version: Video Bulk Editor stores absolutely nothing about you. No database. No logs. No tracking. Your OAuth token lives only in your browser tab and is gone the moment you close it or click Disconnect. Your video data goes directly from your browser to YouTube - our server never sees it.

What data we collect

None. Video Bulk Editor does not collect, store, transmit, or share any personal data, video data, usage data, or analytics of any kind.

YouTube Terms of Service

By using Video Bulk Editor, you agree to be bound by the YouTube Terms of Service. This tool accesses the YouTube Data API v3 and is subject to the YouTube API Services Terms of Service and the Google Privacy Policy.

How authentication works

This tool uses Google OAuth 2.0 to authenticate you with YouTube. Here is exactly what happens:

  • You click "Sign in with Google" and are redirected to Google's own login page. We never see your password.
  • After you approve, Google sends a one-time code to our backend server (hosted on Google Cloud Run).
  • Our server exchanges this code for an OAuth access token and immediately redirects it to your browser via a URL fragment (#token=...). URL fragments are never sent to servers or written to server logs.
  • The token is stored in your browser's sessionStorage only. It is never written to any database, file, or external service.
  • When you close the tab or click Disconnect, the token is deleted from sessionStorage and revoked at Google immediately.

How your video data is handled

All YouTube API calls - reading your video list, reading metadata, saving edits - go directly from your browser to the YouTube Data API v3. Our backend server is never involved in these requests. Your video titles, descriptions, and tags never pass through our server at any point.

Our backend server

Our backend is a small server hosted on Google Cloud Run (Mumbai, India). It handles exactly two things:

  • Redirecting you to Google's OAuth login page
  • Exchanging the one-time OAuth code for a token and forwarding it to your browser

The server has no database connection. It does not log requests. It does not store tokens. The complete source code is publicly readable at github.com/imchikachirag/yt-bulk-editor.

Third parties

No third-party analytics, advertising, tracking, or data-sharing SDKs are used. The only external services this tool communicates with are:

  • Google OAuth 2.0 - for authentication
  • YouTube Data API v3 - for reading and updating your videos
  • Google Cloud Run - hosts the OAuth backend

All three are Google services subject to Google's Privacy Policy.

Cookies

This tool does not use cookies of any kind.

Children's privacy

This tool is intended for YouTube channel owners and managers. It is not directed at children under 13.

Changes to this policy

If this policy changes, the updated date at the top of this page will be updated. Since we collect no data, there is nothing substantive to change.

Revoking access and deleting your data

Since Video Bulk Editor stores no data about you, there is nothing to delete on our end. Your OAuth access token exists only in your browser's sessionStorage for the duration of your session and is automatically removed when you close the tab.

To revoke this tool's access to your YouTube account at any time:

  • Click the Disconnect button inside the app. This immediately removes the token from your browser and revokes it at Google.
  • Or visit your Google Account security settings and remove Video Bulk Editor from the list of connected apps.

Either method fully and permanently removes this tool's access to your YouTube account.

Data Accessed

Video Bulk Editor accesses the following Google user data via the YouTube Data API v3:

  • Your YouTube channel information (channel name, thumbnail, uploads playlist ID)
  • Your YouTube video metadata (titles, descriptions, tags, privacy status, publish dates, view counts, like counts, comment counts, duration, category, language settings)
  • A short-lived OAuth 2.0 access token issued by Google to authenticate API requests on your behalf

The app requests the minimum scopes necessary: youtube.force-ssl for reading and updating video metadata, and youtube.readonly for reading channel and video data.

Data Usage

The data accessed is used solely to display your video library in the editor interface and to apply the title, description, and tag edits you make back to your YouTube channel via the YouTube Data API. No data is used for any other purpose including advertising, analytics, profiling, or training machine learning models.

Data Sharing

Video Bulk Editor does not share any Google user data with any third parties. There are no analytics services, advertising networks, data brokers, or other external services that receive your data. The only external service the app communicates with is the YouTube Data API (googleapis.com) to read and update your video metadata on your behalf.

Data Storage and Protection

Video Bulk Editor stores no Google user data. The OAuth access token is held exclusively in your browser's sessionStorage for the duration of your active session. It is never written to any database, file, server log, or external service. Our backend server (Google Cloud Run) performs only the one-time OAuth code exchange and immediately forwards the token to your browser - it does not retain the token or any user data at any point. The complete server source code is publicly available at github.com/imchikachirag/yt-bulk-editor for independent verification.

Data Retention and Deletion

Since no Google user data is stored by Video Bulk Editor, there is no data to retain or delete on our end. The OAuth access token exists only in your browser's sessionStorage and is automatically removed when you close the browser tab.

To permanently remove this app's access to your Google account:

  • Click the Disconnect button inside the app. This immediately clears the token from your browser and revokes the app's access at Google.
  • Or visit your Google Account security settings and remove Video Bulk Editor from the list of connected apps.
